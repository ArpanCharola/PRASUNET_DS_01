# Data Science Project

This project visualizes the distribution of a categorical or continuous variable, such as the distribution of ages or genders in a population.

## Project Structure

- `data/`: Contains the sample dataset.
- `output/`: Contains the generated visualizations and Excel file.
- `scripts/`: Contains the Python scripts for data processing and visualization.
- `README.md`: Provides an overview of the project.
- `requirements.txt`: Lists the required Python packages.

## How to Run the Project

1. Run the `create_dataset.py` script to create the sample dataset.
2. Run the `create_visualizations.py` script to generate the visualizations.
3. Run the `save_to_excel.py` script to save the data and visualizations to an Excel file.

## Requirements

- pandas
- matplotlib
- seaborn
- openpyxl
